How does it work?

On server side we offer you a REST backend which is build on Node.js and works with MongoDB. Here we store encrypted user credentials, your membership in groups and all the passwords you want to keep safe.
The user interacts with the application on a nice and modern web application build on Angular 4. This application also handles the encryption and decryption which is done asymmetrically using a private and a public key generated and stored by your browser. Therefore the only place where your passwords exists in clear text is your own device.

Getting started

To install passwordManager download the latest release using the button below.
Now you need to extract the .tar.gz file. You will find a folder named build which contains all our code as well as the Docker configuration files. For the next step make sure you have a working installation of Docker. If you are not familiar with Docker or need help with the installation check this page out!
Next you need to navigate to the build directory using your most liked terminal. You will find a init.sh file which needs to be run using bash init.sh. This will start the Docker container and install the passwordManager.
The output of this command will tell you to visit http://localhost:3000/install which is an installation page to create the administrator account. If you are on a mac this page will be opened automatically in your default browser.
After creating your account you can visit http://localhost:3000, log into your new acount and you are ready to go.
